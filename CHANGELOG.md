# Changelog

## 1.0 — 2026-06-27

- 发布首个正式版。
- 修复默认跃迁与战斗速度未在新场景生效的问题。
- 跃迁视觉倍率调整为 1×、2×、5×。
- 驾驶舱航行流光改为由上向下运动。
- 增大 3D 视角远距离敌舰的初始可视尺寸。
- 自动代理会在目标升级缺少记忆且析出后仍可购买时执行记忆析出。
- 移除能源电池资源；记忆析出改为消耗合金与信用点。
- 重绘舰船护盾和离子尾焰，并保留动态脉动与长度变化。
- 2D/3D 战斗共享多敌舰、弹体、激光、导弹与爆炸状态。
- 修复末端补给站离港、改装返回和驾驶舱事件恢复流程。
- 完成商店、自动代理、导弹、随机事件、装备能量和驾驶员系统。
