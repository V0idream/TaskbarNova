# Taskbar Nova：星舰少女

一款常驻 Windows 桌面底部的科幻星舰养成与自动航行游戏。

玩家可以规划星图路线、处理随机事件、探索废弃空间站、收集并改装装备，也可以让两名风格不同的驾驶员接管航行。战斗支持横版战术视角与 3D 驾驶舱视角无缝切换。

## 下载

从 [Releases](https://github.com/Sakuramble/TaskbarNova/releases) 下载 `TaskbarNova-1.0-Windows-x64.exe`，无需安装即可运行。

支持 Windows 10/11 x64。

## 正式版特性

- 半透明、无边框、置顶的桌面常驻窗口。
- 多段星图路线、随机事件、补给站和空间站探索。
- 动能、激光与导弹三类武器，以及分级能量核心、推进器、护盾和雷达。
- 横版与 3D 战斗共享同一套敌舰位置、弹体、命中和爆炸状态。
- 3D 视角支持多敌舰同屏、真实景深弹幕和动态锁定。
- 可摧毁、可机动的分级导弹；动能武器自动预判；激光采用轴线火控。
- `W/S` 手动机动，`A/D` 分别控制两个武器槽；停止操作后恢复自动战术。
- 自动代理可选择多战、避战或均衡路线，并自动整备、出售低阶装备和合理析出记忆。
- 装备受舰船能量容量限制；高级商店物品需要信用点、合金和记忆。
- Nova 与 Lumi 拥有不同对话风格和驾驶员技能。
- 鼠标穿透、窗口透明度、默认跃迁/战斗速度、折叠模式等桌面功能。

## 操作

| 功能 | 操作 |
| --- | --- |
| 战斗移动 | `W` / `S` |
| 武器槽 1 / 2 | `A` / `D` |
| 鼠标穿透 | `Ctrl+Alt+P` 或 `F8` |
| 退出手动战斗 | 停止操作 1 秒 |

跃迁计时支持 `1× / 4× / 12×`；对应视觉动画速度为 `1× / 2× / 5×`，避免高倍模式过度闪烁。

## 开发

```powershell
pnpm install
pnpm run dev
pnpm run build
pnpm run dist
```

技术栈：Electron、React、TypeScript、Vite、Zustand。

## 存档兼容

游戏会自动迁移旧版本存档。正式版已移除“能源电池”资源；旧存档中的该字段会在读取时安全忽略，舰船能量核心与装备能耗系统不受影响。
